# SkillsHub-OnlineLearning-Platform

The SkillHub is an online self-paced learning management system designed to offer a wide range of short courses to learners. It allows users to enroll in and access various courses, complete assignments, and undergo assessments flexibly. The primary focus of this project is to develop a user-friendly, web-based platform that caters to both learners and instructors, streamlining the process of course creation, enrollment, and completion.
